Requisitos 'Formulário 3 - Novos Campos'

Campos

Upload de Arquivo (PDF)

  • Requisito: Usuário deve selecionar um arquivo em formato PDF.
  • Critérios de aceite:
    • Campo obrigatório.
    • Somente arquivos com extensão .pdf são aceitos.
    • Mensagem exibida: “Somente arquivos PDF são permitidos.”

Upload de Arquivo (DOCX)

  • Requisito: Usuário deve selecionar um arquivo em formato DOCX.
  • Critérios de aceite:
    • Campo obrigatório.
    • Somente arquivos com extensão .docx são aceitos.
    • Mensagem exibida: “Somente arquivos DOCX são permitidos.”

Upload de Arquivo (JPG)

  • Requisito: Usuário deve selecionar uma imagem em formato JPG.
  • Critérios de aceite:
    • Campo obrigatório.
    • Somente arquivos com extensão .jpg são aceitos.
    • Mensagem exibida: “Somente arquivos JPG são permitidos.”

Seleção de País, Estado e Cidade

  • Requisito: Usuário deve selecionar um país, um estado e uma cidade.
  • Critérios de aceite:
    • Todos os três campos são obrigatórios.
    • Não deve permitir envio se algum campo estiver vazio.
    • As opções devem aparecer organizadas em hierarquia (País → Estado → Cidade).
    • Mensagem exibida: “Preencha corretamente os campos de País, Estado e Cidade.”

Botões

Enviar

  • Requisito: Executa a validação dos campos e simula envio do formulário.
  • Critérios de aceite:
    • Se todos os campos obrigatórios estiverem corretos, exibir mensagem de sucesso.
    • Mensagem exibida: “Formulário enviado com sucesso!”.
    • Não deve permitir envio com campos obrigatórios vazios ou arquivos inválidos.

Voltar

  • Requisito: Redireciona para a tela Home.
  • Critério de aceite: Ao clicar, deve carregar home.html.

  • Critério de aceite: Ao clicar, deve carregar formulario-3-requisitos.html.
