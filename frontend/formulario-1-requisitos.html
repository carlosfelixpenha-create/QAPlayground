Requisitos 'Formulário 1 - Cadastro de Endereço'

Campos

Logradouro

  • Requisito: Deve permitir digitar texto (rua, avenida, etc.).
  • Critérios de aceite:
    • Campo não pode estar vazio.
    • Não deve conter apenas números.
    • Exibir mensagem de erro se não atender às regras.
    • Mensagem exibida: “Preencher corretamente o campo Logradouro, dúvida entrar em requisitos!”

Número

  • Requisito: Deve permitir apenas números.
  • Critérios de aceite:
    • Campo não pode estar vazio.
    • Não deve aceitar letras ou caracteres especiais.
    • Mensagem exibida: “Preencher corretamente o campo Número, dúvida entrar em requisitos!”

Complemento

  • Requisito: Campo opcional para informações adicionais.
  • Critérios de aceite:
    • Pode estar vazio.
    • Deve aceitar texto livre.

Bairro

  • Requisito: Deve permitir texto (mínimo 3 caracteres).
  • Critérios de aceite:
    • Campo não pode estar vazio.
    • Não deve aceitar números isolados.
    • Mensagem exibida: “Preencher corretamente o campo Bairro, dúvida entrar em requisitos!”

Cidade

  • Requisito: Deve permitir texto (mínimo 3 caracteres).
  • Critérios de aceite:
    • Campo não pode estar vazio.
    • Não deve aceitar números isolados.
    • Mensagem exibida: “Preencher corretamente o campo Cidade, dúvida entrar em requisitos!”

Estado

  • Requisito: Deve permitir apenas 2 letras (UF).
  • Critérios de aceite:
    • Campo não pode estar vazio.
    • Deve aceitar apenas letras maiúsculas.
    • Mensagem exibida: “Preencher corretamente o campo Estado, dúvida entrar em requisitos!”

CEP

  • Requisito: Deve permitir apenas números (8 dígitos).
  • Critérios de aceite:
    • Campo não pode estar vazio.
    • Não deve aceitar letras ou caracteres especiais.
    • Mensagem exibida: “Preencher corretamente o campo CEP, dúvida entrar em requisitos!”

Botões

Salvar

  • Requisito: Executa a validação dos campos e simula cadastro de endereço.
  • Critérios de aceite:
    • Se todos os campos obrigatórios estiverem corretos, exibir mensagem de sucesso.
    • Mensagem exibida: “Endereço cadastrado com sucesso!”.
    • Não deve permitir salvar com campos obrigatórios vazios.

Voltar

  • Requisito: Redireciona para a tela Home.
  • Critério de aceite: Ao clicar, deve carregar home.html.
