Requisitos 'Formulário 2 - Campos Diversos'

Campos

Sexo (Radio)

  • Requisito: Usuário deve selecionar uma opção (Masculino ou Feminino).
  • Critérios de aceite:
    • Campo obrigatório, não pode ficar sem seleção.
    • As opções devem aparecer alinhadas ao lado do seletor (input + label juntos).
    • Mensagem exibida: “Selecione uma opção no campo Sexo, dúvida entrar em requisitos!”

Interesses (Checkbox)

  • Requisito: Usuário pode marcar múltiplos interesses (Frontend, Backend, QA).
  • Critérios de aceite:
    • Campo opcional, pode estar vazio.
    • Se marcado, deve exibir corretamente os valores escolhidos.
    • As opções devem aparecer alinhadas ao lado do seletor (input + label juntos).

Data de Nascimento

  • Requisito: Deve permitir selecionar uma data válida.
  • Critérios de aceite:
    • Campo obrigatório.
    • Não deve aceitar datas futuras.
    • Mensagem exibida: “Preencher corretamente o campo Data de Nascimento, dúvida entrar em requisitos!”

Telefone

  • Requisito: Deve permitir digitar telefone com máscara (formato (99) 99999-9999).
  • Critérios de aceite:
    • Campo obrigatório.
    • Deve conter apenas números válidos no padrão brasileiro.
    • Mensagem exibida: “Preencher corretamente o campo Telefone, dúvida entrar em requisitos!”

CPF

  • Requisito: Deve permitir digitar CPF com máscara (formato 999.999.999-99).
  • Critérios de aceite:
    • Campo obrigatório.
    • Deve conter exatamente 11 dígitos numéricos.
    • Não deve aceitar letras ou caracteres especiais fora da máscara.
    • Mensagem exibida: “Preencher corretamente o campo CPF, dúvida entrar em requisitos!”

Botões

Enviar

  • Requisito: Executa a validação dos campos e simula envio do formulário.
  • Critérios de aceite:
    • Se todos os campos obrigatórios estiverem corretos, exibir mensagem de sucesso.
    • Mensagem exibida: “Formulário enviado com sucesso!”.
    • Não deve permitir envio com campos obrigatórios vazios.

Voltar

  • Requisito: Redireciona para a tela Home.
  • Critério de aceite: Ao clicar, deve carregar home.html.

Requisitos

  • Requisito: Redireciona para a tela de requisitos.
  • Critério de aceite: Ao clicar, deve carregar formulario-2-requisitos.html.
